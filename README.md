Escape from Miami is a mod for Hotline Miami 2: Wrong Number that overhauls the vanilla gun sound effects with ones from Escape from Tarkov. 

Video showcase: [here](https://youtu.be/4BeK4rYsADU)

## INSTALLATION INSTRUCTIONS:
1. Navigate to C:\Users\[your username]\Documments
2. Within the Documents folder, find "My Games"
3. Open the folder "HotlineMiami2" within "My Games"
4. Once inside the HotlineMiami2 folder, open "mods" 
5. Place "escape_from_miami_sfx.patchwad" inside the folder (Remove any other sounds mods inside this folder, as they will NOT BE COMPATIBLE if they change the same weapon sounds that this mod does!)
6. Done!
